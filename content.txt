RAIR:
RAIR introduces an LLM reasoning layer that interprets user edits, 
retrieves contextual knowledge through RAG, 
and guides iterative query refinement for interactive image retrieval.

Rag có thể thu thập:
- candidate images
- scene graph patterns
- dataset knowledge

LLM dùng thông tin này để reasoning
- analyze missmatch
- propose edit
- explain result

LLM đóng vai trò:
Reasoning + Query Planning + Interaction Assistance
Các Modules:
- Query Interpretation
convert free-text -> triplets
- Query Diagnosis (Reasoning)
sử dụng kết quả của RAG để:
	+ xác định missmatch
	+ xác định chủ đề
	+ xác định đề xuất chỉnh sửa
	+ trả kết quả của turn hiện tại
- Query Refinement
Đề xuất chỉnh sửa
Giải thích kết quả:
	- Lý do tại sao chọn (interpretability - khả năng giải thích)
	- Đưa ra các gợi ý tối thiểu (interaction guidance)
User có thể đồng ý, từ chối, hoặc sửa theo ý họ

-> lặp tiếp
=> trả về sẽ là:
- triplet
- suggestions

User input 
-> [Prompt 1] Convert sang triplets 
-> Retrieve (RAG) 
-> [Prompt 2] Suy luận kết quả từ RAG 
-> [Prompt 3] Đề Xuất chỉnh sửa, giải thích 
-> User Edit 
Lặp lại
